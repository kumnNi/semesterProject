# 🎓 Studienbewerbungssystem

Ein Webprojekt für die **Online-Bewerbung auf Studienplätze**, entwickelt mit **JavaScript** und **React**.
Bewerber können Studiengänge sich online bewerben und ihre Bewerbungen verwalten.
Das System bietet eine moderne und benutzerfreundliche Oberfläche für den gesamten Bewerbungsprozess.

---

## 🚀 Features

* Benutzerregistrierung und Login
* Studiengänge anzeigen
* Online-Bewerbung einreichen
* Admin-Bereich zur Verwaltung von Bewerbungen und Benutzern
* Responsives Design für Mobile und Desktop

---

## 🛠 Technologien

* JavaScript
* React
* Bootstrap
* MongoDB

---


## 📌 Verwendung

* Benutzer registrieren sich und melden sich an
* Studiengänge durchsuchen
* Bewerbung online ausfüllen und absenden
* Administratoren verwalten Bewerbungen und Benutzer
